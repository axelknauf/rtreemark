# TreeMarkable

> Remember + Mark + Remark = Remarkable

Mark remarkable trees in order to remember them.

## Requirements

Rust and Cargo:

* https://www.rust-lang.org/tools/install

## Build, Test & Run

    # development build
    cargo build

    # production build
    cargo build --release

    # run tests
    cargo test

    # run during development
    cargo run
